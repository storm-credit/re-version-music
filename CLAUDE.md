# Re:Version Music

감정 재해석 음악 채널 프로젝트. 원곡의 감정 구조를 추출해 시점·문화·장르를 변환한 오리지널 곡을 만든다.

## 채널 컨셉

"같은 감정, 다른 버전" — 커버가 아닌 감정 번안·재해석 채널

## 콘텐츠 유형 (4가지)

1. **번안형** (40%) — 팝송 감성을 한국식으로, 또는 한국곡을 해외 감성으로
2. **답가형** (30%) — 원곡 화자의 상대방이 답하는 곡
3. **팬픽OST형** (20%) — 가상 캐릭터/세계관 테마곡
4. **오리지널형** (10%) — 완전 오리지널 실험곡

## 제작 파이프라인

1. 원형 추출: 화자, 상대, 사건, 핵심 감정
2. 변환 규칙: 언어/문화/시점/장르 변환 방향 결정
3. 후렴 우선 제작: 쇼츠용 한 방 후렴
4. 가사 작성: 1절 → 2절 → 브릿지
5. Suno 프롬프트 작성 및 생성
6. 영상 패키징: 상황형 영상 + 자막

## 워크플로우

- 곡 기획은 `02. 곡 기획/` 하위 유형별 폴더에 작성
- 가사·프롬프트·편곡노트는 `03. 제작/`에 저장
- 영상 스크립트와 썸네일 기획은 `04. 영상/`에 저장
- 발행 후 기록은 `05. 발행/`에 남김
- 반복 사용 템플릿은 `06. 템플릿/`에 보관

## 저작권 원칙

- 원곡 멜로디·가사 직접 사용 금지
- "inspired by" / "reimagined as" 방식
- 감정·상황·서사만 차용, 음악은 오리지널
- 특정 IP 캐릭터 직접 복제 금지
