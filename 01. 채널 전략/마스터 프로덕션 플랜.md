# Re:Version Music 마스터 프로덕션 플랜

## 원칙
- 모든 곡은 **특정 원곡/IP를 명시**한다
- 음악은 100% 오리지널 (멜로디·가사 새로 제작)
- 영상 제목과 설명에 "inspired by" 명시
- 원곡의 **감정 구조만** 차용

### 번안형 제작 흐름
```
1. 원곡 감정 분석 (화자/상대/사건/감정)
2. 문화 차이점 정리 (미국은 이렇고, 한국은 저렇다)
3. 한국 정서로 변환된 후렴 작성
4. Suno 프롬프트 (장르 변환 반영)
5. 이미지: 원문화 vs 한국 대비 비주얼
6. 쇼츠: 원곡명 언급 → "이 감정이 한국이었다면" → 후렴
```

## 답가형 일반 10곡 — "이 노래에 답장을 보낸다면"

| # | 곡 제목 | 원곡 (답가 대상) | 원곡 화자 → 답가 화자 | 쇼츠 훅 |
|---|---------|----------------|---------------------|---------|
| 1 | 사실 못 보냈던 말 | 태양 — 눈, 코, 입 | 남겨진 남자 → **떠난 여자** | "태양 '눈, 코, 입'에 대한 떠난 그녀의 답가" |
| 2 | 미안해가 사랑이었어 | 에디킴 — 이쁘다니까 | 직진 고백 남자 → **거절한 여자** | "에디킴 '이쁘다니까' 고백에 대한 거절한 그녀의 답가" |
| 3 | 거짓말이야 나도 | 다비치 — 8282 | 이별 후 후회 → **통보받은 상대** | "다비치 '8282' 후회에 대한 상대의 답가" |
| 4 | 기다리지 마 제발 | 이소라 — 바람이 분다 | 남겨진 여자 → **떠난 사람** | "이소라 '바람이 분다'에 대한 떠난 사람의 답가" |
| 5 | 알고 있었어 | 성시경 — 거리에서 | 짝사랑 남자 → **알고 있던 여자** | "성시경 '거리에서' 짝사랑에 대한 상대의 답가" |
| 6 | 네가 모르는 이유 | 버즈 — 가시 | 상처받은 남자 → **떠난 여자 해명** | "버즈 '가시' 원망에 대한 떠난 사람의 답가" |
| 7 | 축하해 진심으로 (거짓말) | 이승기 — 결혼해줄래 | 프로포즈 남자 → **전 연인 하객** | "'결혼해줄래'를 듣는 전 연인의 노래" |
| 8 | 나도 보고싶었어 | 김범수 — 보고싶다 | 그리워하는 남자 → **떠난 상대** | "김범수 '보고싶다'에 대한 떠난 상대의 답가" |
| 9 | 힘내라는 말이 제일 힘들어 | 이하이 — 한숨 | 위로하는 사람 → **위로받는 사람** | "이하이 '한숨' 위로에 대한 솔직한 답가" |
| 10 | 쫓아갈걸 | 지드래곤 — 그XX | 자존심 이별 → **6개월 후 후회** | "지드래곤 '그XX' 자존심의 대가, 6개월 후 답가" |

### 답가형 제작 흐름
```
1. 원곡 분석: 화자/상대/사건/감정
2. 반대 시점 설정: 원곡의 상대방은 어떤 감정이었을까
3. 대비표 작성: 원곡 "잘가" → 답가 "사실 못 보냈어"
4. 후렴 먼저 작성 (원곡의 핵심 메시지에 대한 답)
5. Suno 프롬프트 (원곡과 다른 장르도 OK)
6. 이미지: 원곡과 같은 상황, 반대 시점 비주얼
7. 쇼츠: 원곡 상황 설명 → "상대방은 이렇게 생각했다" → 후렴
```

## 팬픽OST 5곡 — "이 캐릭터의 테마곡"

| # | 곡 제목 | 원작 IP (영감) | 캐릭터/상황 | 쇼츠 훅 |
|---|---------|-------------|-----------|---------|
| 1 | 이번엔 다르게 죽을게 | 악녀는 모래시계를 되돌린다 (웹툰) | 회귀한 악녀가 복수 대신 자유를 택하는 순간 | "회귀한 악녀가 이번엔 도망친다" |
| 2 | 악역이 아니었어 | 나 혼자만 레벨업 (웹소설/애니) | 최종보스가 사실 주인공을 키운 스승이었다면 | "최종보스가 쓰러지며 웃은 이유" |
| 3 | 몸이 먼저 기억했어 | 도깨비 (드라마) | 기억을 잃은 연인이 본능으로 다시 사랑에 빠지는 순간 | "기억을 잃어도 사라지지 않는 사랑" |
| 4 | 네온 불빛 아래 마지막 담배 | 사이버펑크: 엣지러너 (애니) | 지친 현상금 사냥꾼의 은퇴 전날 밤 | "네온 도시를 떠나는 사냥꾼의 마지막 밤" |
| 5 | 2006년의 나에게 | 시그널 (드라마) | 시간여행자가 과거의 자신에게 보내는 편지 | "20년 후의 내가 편지를 보낸다면" |

### 팬픽OST 제작 흐름
```
1. 원작 IP 핵심 장면/감정 분석
2. 캐릭터 archetype + 관계성 + 사건 정의
3. 스토리 한 줄 요약
4. 후렴 먼저 (캐릭터의 감정 폭발 순간)
5. Suno 프롬프트 (세계관에 맞는 장르)
6. 이미지: 원작 직접 안 쓰고, 같은 분위기 오리지널 비주얼
7. 쇼츠: 캐릭터 소개 → 관계 → 감정 → 후렴
```

### 팬픽OST 저작권 주의
- 원작 캐릭터 이름/고유명사 직접 사용 ✗
- "회귀한 악녀" "최종보스" 등 아키타입으로 표현 ✓
- 영상에 원작 이미지/영상 사용 ✗
- AI 생성 오리지널 이미지만 사용 ✓
- 설명란에 "inspired by ___" 표기 ✓
