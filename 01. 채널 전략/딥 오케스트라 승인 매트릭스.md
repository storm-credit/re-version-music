# 딥 오케스트라 승인 매트릭스

이 매트릭스는 오케스트라 총괄이 전문가 의견을 모아 최종 승인/조건부 승인/보류를 결정할 때 사용한다.

## 점수 기준

| 점수 | 의미 |
|---:|---|
| 0 | 없음. 지금 상태로 진행하면 안 됨 |
| 1 | 약함. 방향은 있으나 재작업 필요 |
| 2 | 통과. 제작 가능하지만 보강 여지 있음 |
| 3 | 강함. 바로 제작/발행 가능 |

## 필수 평가 항목

| 항목 | 질문 | 0점 | 1점 | 2점 | 3점 |
|---|---|---|---|---|---|
| 감정 원형 | 화자/상대/사건/핵심 감정이 분명한가? | 없음 | 감정만 있음 | 사건까지 있음 | 후렴 문장까지 선명 |
| 변환축 | 메인 변환축이 하나인가? | 없음 | 여러 개 충돌 | 메인/보조 분리 | 한 줄로 설명 가능 |
| 후렴 훅 | 첫 줄이 기억되는가? | 없음 | 설명적 | 자막으로 이해 | 제목처럼 기억 |
| 음악 지시 | Suno 프롬프트가 안전하고 구체적인가? | 없음 | 장르만 있음 | 보컬/악기/BPM 있음 | 선택 기준까지 있음 |
| 비주얼 | 첫 화면과 후렴 장면이 감정을 보여주는가? | 없음 | 예쁘기만 함 | 감정 흐름 있음 | 썸네일까지 강함 |
| 저작권 안전 | 원곡/IP 복제 위험이 낮은가? | 높음 | 중간 이상 | 낮음 | 안전 문구까지 완료 |
| 발행력 | 제목/설명/고정댓글/지표가 준비됐는가? | 없음 | 제목만 있음 | 패키지 있음 | 테스트 기준까지 있음 |
| 아카이브 | 다음 작업자가 이어받을 수 있는가? | 없음 | 파일만 있음 | 위치 기록 | 재사용 패턴 기록 |

## 판정

| 총점 | 판정 | 처리 |
|---:|---|---|
| 0~11 | 보류 | 감정 원형 또는 변환축부터 재작업 |
| 12~17 | 조건부 승인 | 부족한 전문가 라운드만 재실행 |
| 18~21 | 승인 | 제작 진행 가능 |
| 22~24 | 강승인 | 제작/발행 우선순위 상향 |

## 강제 보류 조건

총점과 무관하게 아래 항목이 있으면 보류한다.

- 원곡 가사 또는 멜로디를 직접 사용한다.
- 특정 아티스트의 목소리/사운드를 복제하라고 지시한다.
- 공식 발매물, 공식 MV, 공식 팬송처럼 보인다.
- 원작 캐릭터명, 고유 대사, 외형을 그대로 사용한다.
- 후렴이 없거나 후렴 첫 줄이 곡의 감정을 말하지 못한다.

## 유형별 가중 판단

### 번안형

가중 항목:
- 변환축
- 문화 클리셰 위험
- 후렴 훅

보류 신호:
- 원곡명만 빼면 설명이 안 된다.
- 문화권이 음식/의상/랜드마크로만 표현된다.

### 답가형

가중 항목:
- 원곡과 답가의 논리
- 화자/상대 관계
- 저작권 안전

보류 신호:
- 원곡을 단순 반박만 한다.
- 상대 화자의 새 정보가 없다.

### 팬픽OST형

가중 항목:
- 아키타입 선명도
- 관계성
- IP 복제 위험

보류 신호:
- 원작 캐릭터 이름과 외형에 의존한다.
- 세계관 설명은 많은데 감정 사건이 없다.

### 오리지널형

가중 항목:
- 상황 구체성
- 후렴 훅
- 발행력

보류 신호:
- 분위기는 있는데 사건이 없다.
- 실험성이 감정 전달을 가린다.
