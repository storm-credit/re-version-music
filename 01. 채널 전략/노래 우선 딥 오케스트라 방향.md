# 노래 우선 딥 오케스트라 방향

## 1. 우리가 만드는 것

Re:Version Music은 커버, 리믹스, 한국판 제작 채널이 아니다.

```text
같은 감정, 다른 버전.
좋은 곡의 작동 방식을 연구하되, 결과물은 완전한 오리지널 곡으로 만든다.
```

원곡에서 가져올 수 있는 것은 감정 구조, 사건의 압력, 후렴이 기억되는 방식, 장르적 질감이다. 가져오면 안 되는 것은 멜로디, 가사, 고유 리프, 특정 보컬, 공식 캐릭터 외형, MV 장면이다.

## 2. 최우선 판단

노래가 먼저다. 영상, 캐릭터, 썸네일은 좋은 노래를 더 잘 보이게 만드는 장치다.

| 우선순위 | 질문 | 실패 신호 |
|---|---|---|
| 1 | 후렴이 다시 듣고 싶은가 | 콘셉트는 좋은데 노래가 약함 |
| 2 | 화자의 감정이 선명한가 | 분위기만 있고 사건이 없음 |
| 3 | 장르가 감정을 밀어주는가 | 여러 장르를 섞어 정체성이 흐림 |
| 4 | 원곡 없이도 독립적으로 서는가 | 참조곡 이름을 빼면 매력이 사라짐 |
| 5 | Suno가 잘 생성할 수 있는가 | 지시가 넓고 모순됨 |
| 6 | 캐릭터/화자와 스토리보드가 일관적인가 | 컷마다 인물과 감정이 흔들림 |
| 7 | 롱폼 MV로 끝까지 볼 수 있는가 | 장면은 예쁜데 곡과 따로 놂 |

## 3. 좋은 곡 모티브 사용법

좋은 곡을 모티브로 삼을 때는 아래만 추출한다.

```text
감정 엔진:
후렴 작동 방식:
에너지 곡선:
보컬 거리감:
편곡 질감:
섹션 역할:
반복 청취 이유:
```

반드시 버린다.

```text
멜로디:
가사 문장:
고유 리프:
시그니처 드롭:
아티스트 보컬 모사:
공식 MV 장면:
```

## 4. 장르 운영

장르는 여러 개를 운영하되, 곡 하나에는 중심 장르를 하나만 둔다.

| 장르 레인 | 쓰기 좋은 감정 | 대표 변환 |
|---|---|---|
| Korean dawn ballad | 미련, 말 못 한 고백 | 직진 감정 -> 삼킨 말 |
| R&B / neo soul | 후회, 욕망, 거리감 | 말보다 호흡이 앞서는 관계 |
| city pop | 도시적 추억, 늦은 밤 | 밝은 그루브 안의 쓸쓸함 |
| indie rock | 청춘의 자조, 무력감 | 감정을 소리 내지 못하는 폭발 |
| synthpop | 거리, 기억, 비현실 | 차가운 빛 속의 고독 |
| bossa / acoustic | 체념, 오래된 그리움 | 부드럽게 무너지는 감정 |
| trot / retro pop | 서러움, 체면, 해학 | 울음을 웃음으로 숨김 |
| pansori-pop | 한, 선언, 운명 | 개인 감정을 의식처럼 확장 |
| ambient electronic | 꿈, 기억, 상실 | 사건보다 감각이 남는 곡 |
| cinematic pop | 큰 선택, 클라이맥스 | 마지막 후렴의 해방감 |

```text

11. 컷별 비디오 프롬프트

```

## 7. 강제 보류 조건

아래 중 하나라도 있으면 총괄은 보류한다.

- 참조곡 멜로디를 따라가려는 지시가 있다.
- 원곡 가사를 번역하거나 구조만 바꿔 쓰고 있다.
- 특정 아티스트 목소리를 요구한다.
- IP 캐릭터의 이름/외형/대사를 직접 쓴다.
- 곡보다 썸네일/세계관이 먼저 커졌다.
- 장르가 3개 이상 전면에 나와 Suno 지시가 흐려졌다.
