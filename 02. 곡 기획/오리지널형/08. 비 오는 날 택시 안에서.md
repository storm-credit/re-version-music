# 08. 비 오는 날 택시 안에서

## 콘셉트
- **주제**: 비 오는 날 택시 뒷좌석에서 느끼는 고요한 안식
- **화자**: 퇴근 후 택시를 탄 직장인
- **상황**: 비가 쏟아지는 저녁, 택시 뒷좌석에서 창밖을 바라보며
- **핵심 감정**: 고요한 위안, 움직이면서 멈춰 있는 듯한 평화
- **한 줄 요약**: 미터기가 올라가는 동안만큼은 아무도 나를 찾지 않는다

## 쇼츠 훅
"이 택시가 영원히 달렸으면 좋겠어"

---

## Suno 프롬프트

### Style of Music

```
[Genre]: Jazz-influenced pop
[Mood]: Serene, rainy, contemplative, cozy, floating
[Tempo]: 78 BPM
[Instruments]: Nylon string guitar, muted trumpet, upright bass (walking), brush drums on snare, warm Rhodes piano, rain ambient texture, soft vibraphone, subtle string pad
[Vocal]: Male vocal, smooth jazz-influenced, intimate and velvety
[Structure]: Intro (rain + Rhodes) → Verse 1 (guitar + voice) → Verse 2 (add bass, brushes) → Pre-Chorus (trumpet enters) → Chorus (full jazz combo) → Verse 3 (intimate, pull back) → Pre-Chorus → Chorus → Bridge (trumpet solo + vocal) → Final Chorus (warm, full) → Outro (rain fade)
[Language]: Korean
[Theme]: 빗소리에 감싸인 택시 안의 고요가 주는 예상치 못한 평화
```

## 영상 콘셉트
- **메인 이미지**: 비 내리는 창밖의 흐릿한 도시 불빛, 택시 뒷좌석 시점
- **색감**: 빗물에 번진 네온 불빛 (빨강, 노랑, 파랑) + 택시 안 따뜻한 앰버
- **키 장면 3개**:
  1. 택시 창에 흐르는 빗방울 사이로 보이는 흐릿한 도시 불빛
  2. 김 서린 창에 손가락으로 동그라미를 그리는 장면
  3. 택시에서 내려 우산을 펴는 순간, 뒤돌아보는 모습

## 썸네일 방향
- 비에 젖은 택시 창문 너머로 보이는 흐릿한 도시 불빛, 빗방울에 맺힌 보케 효과. "택시 안에서" 텍스트를 김 서린 창에 쓴 듯한 느낌으로

## 메모
- BPM: 78
- Key: Db Major
- 보컬: 남성 보컬, 부드럽고 재즈적인 톤, 벨벳 같은 질감
