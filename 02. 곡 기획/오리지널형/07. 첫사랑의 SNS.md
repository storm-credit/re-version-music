# 07. 첫사랑의 SNS

## 콘셉트
- **주제**: 우연히 마주친 첫사랑의 SNS에서 느끼는 복잡한 감정
- **화자**: 첫사랑의 인스타를 우연히 발견한 사람
- **상황**: 늦은 밤, 추천 피드에 뜬 첫사랑의 행복한 사진을 보며
- **핵심 감정**: 그리움, 안도, 씁쓸함이 뒤섞인 묘한 감정
- **한 줄 요약**: 잘 지내고 있구나, 라는 말이 반갑기도 아프기도 해

## 쇼츠 훅
"팔로우는 안 했어, 그냥 보기만 했어"

### Style of Music

```
[Genre]: Dreamy indie pop
[Mood]: Wistful, hazy, bittersweet, floaty, melancholic-warm
[Tempo]: 88 BPM
[Instruments]: Jangly electric guitar, dreamy synth pad, shimmering reverb guitar, soft programmed drums, melodic bass, vibraphone, tape delay vocals, subtle shaker
[Vocal]: Female vocal, airy and dreamy, with nostalgic breathiness
[Structure]: Intro (shimmery guitar) → Verse 1 (floaty, sparse) → Verse 2 (add rhythm) → Pre-Chorus (build shimmer) → Chorus (dreamy full band) → Verse 3 (intimate) → Pre-Chorus → Chorus → Bridge (stripped, vulnerable) → Final Chorus (layered harmonies) → Outro (guitar shimmer fade)
[Language]: Korean
[Theme]: 지나간 사랑의 흔적을 디지털 세계에서 마주하는 감정
```

## 영상 콘셉트
- **메인 이미지**: 어두운 방에서 인스타그램 피드를 스크롤하는 손, 화면 빛에 비친 얼굴
- **색감**: 인스타그램 그라데이션 (보라-핑크-오렌지) + 어두운 네이비
- **키 장면 3개**:
  1. SNS 피드에서 첫사랑의 사진이 나타나는 순간 손가락이 멈추는 장면
  2. 과거 회상 — 같은 카페에서 마주 앉은 두 사람의 실루엣
  3. 핸드폰을 내려놓고 창밖을 바라보는 옆모습

## 썸네일 방향
- 인스타그램 프로필 화면을 모티브로, 흐릿한 프로필 사진과 "팔로우" 버튼 위에 망설이는 손가락. "첫사랑의 SNS" 텍스트를 인스타 UI 느낌으로
