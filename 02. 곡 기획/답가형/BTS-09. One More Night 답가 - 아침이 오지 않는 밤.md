# BTS-09. 아침은 약속을 안 지켜 (One More Night 답가)

## 원곡 분석
- **원곡**: BTS - One More Night
- **메시지**: 판타지 같은 이 밤을 붙잡고 싶다, 하룻밤만 더
- **감정**: 환상, 집착, 아쉬움

- **답가 화자**: "하룻밤만 더"를 빌었지만 아침이 와버린 사람

- **대비**: 원곡 '밤을 붙잡자'면, 답가는 '아침이 깨운 현실'

## 쇼츠 훅

## 영상 제작 가이드

### 이미지 프롬프트
```
Scene 1: Alarm clock showing 7:00AM, harsh morning light cutting through curtains
Scene 2: Empty pillow next to sleeping person, indent still visible, ghost of presence
Scene 3: Two coffee cups → one put back in cabinet, kitchen morning light, lonely
Scene 4: Mirror reflection, morning face, reality hitting, raw
Scene 5: Same room night vs morning split - warm/dreamy vs cold/harsh
Scene 6: Sunset beginning again, cycle restarting, purple to gold
```

### 영상 구성
- 색감 패턴: 따뜻(밤 회상) ↔ 차가운(아침 현실) 반복
- 효과: 밤 장면 = 블러/글로우 / 아침 장면 = 샤프/차가운
- 핵심: 브릿지에서 "밤이 진짜 아닐까" 순간 두 톤 겹침

### 썸네일
- 반반: 따뜻한 밤 vs 차가운 아침 + "One More Night의 다음 날"

- BPM: 100
- Key: B minor
