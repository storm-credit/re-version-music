# 저작권-감정일관성 리뷰 체크리스트

이 체크리스트는 곡 기획, 가사, Suno 프롬프트, 영상 프롬프트, 업로드 패키지에 모두 적용한다.

## 1. 저작권 안전선

- [ ] 원곡 가사를 직접 사용하지 않았다.
- [ ] 원곡 멜로디를 재현하도록 지시하지 않았다.
- [ ] 원곡의 제목/후렴 구조를 과하게 닮지 않았다.
- [ ] 특정 아티스트의 보컬을 복제하라고 지시하지 않았다.
- [ ] 원작 캐릭터의 이름, 고유 대사, 외형을 그대로 사용하지 않았다.
- [ ] 원작 이미지, 영상, 로고 사용을 전제로 하지 않았다.
- [ ] 설명란의 `inspired by` 표기가 복제 허가처럼 보이지 않는다.

## 2. 감정 원형 일관성

- [ ] 화자가 분명하다.
- [ ] 상대가 분명하다.
- [ ] 사건이 분명하다.
- [ ] 핵심 감정이 한 문장으로 설명된다.
- [ ] 후렴이 핵심 감정과 같은 방향이다.
- [ ] 영상 배경이 같은 감정을 보여준다.
- [ ] 제목이 본편보다 과장된 다른 약속을 하지 않는다.

## 3. 유형별 리뷰

### 번안형

- [ ] 문화 변환이 고정관념만으로 처리되지 않았다.
- [ ] 원문화와 변환문화의 감정 표현 차이가 보인다.
- [ ] 장르 변환이 감정을 흐리지 않는다.
- [ ] `이 곡이 한국/해외였다면`이라는 질문에 답한다.

### 답가형

- [ ] 원곡 화자와 답가 화자의 관계가 논리적으로 이어진다.
- [ ] 답가가 단순 반박이 아니라 새로운 감정 정보를 준다.
- [ ] 원곡 팬이 알아들을 입구가 있다.
- [ ] 원곡을 몰라도 한 편의 짧은 이야기로 이해된다.

### 팬픽OST형

- [ ] 캐릭터는 고유명사보다 아키타입으로 설명된다.
- [ ] 원작 외형을 복제하지 않는다.
- [ ] 세계관 톤이 장면, 가사, 음악에 함께 반영된다.
- [ ] 캐릭터의 선택 또는 상처가 후렴에 드러난다.

### 오리지널형

- [ ] 상황 훅이 충분히 구체적이다.
- [ ] 일상 사물 또는 장소가 감정을 붙잡아준다.
- [ ] 실험성이 후렴 전달력을 방해하지 않는다.

## 4. Suno 프롬프트 리뷰

- [ ] 장르, BPM, 보컬, 악기, 무드가 구체적이다.
- [ ] Style of Music 키워드가 5~8개 안에서 정리된다.
- [ ] 특정 곡명이나 아티스트명을 사운드 복제 지시로 쓰지 않았다.
- [ ] 후렴이 먼저 테스트 가능한 구조다.
- [ ] 생성 후 선택 기준이 적혀 있다.

## 5. 영상/썸네일 리뷰

- [ ] 첫 1초 훅이 있다.
- [ ] 썸네일은 사물 1개, 감정 1개, 짧은 텍스트 중심이다.
- [ ] 자막은 2줄 이내로 읽힌다.
- [ ] 장면 전환이 감정 변화와 맞는다.
- [ ] 인물/배경이 원작을 직접 복제하지 않는다.

## 6. 승인 판단

승인 상태: 승인 / 조건부 승인 / 보류
다음 액션:
