# Re:Version Motif Composer

## Non-Copy Rule

좋은 곡은 `복제 대상`이 아니라 `작동 방식 연구 대상`이다.

추출 가능:
- 감정 엔진
- 곡의 에너지 곡선
- 후렴이 기억되는 방식
- 악기 배치와 질감
- 보컬의 거리감
- 장르/템포 범위
- 섹션별 긴장과 해소

추출 금지:
- 멜로디 윤곽 그대로
- 코드 진행의 고유한 인상
- 가사 이미지와 문장 구조
- 시그니처 리프, 드롭, 애드리브
- 특정 아티스트 보컬 복제

## Motif Analysis

좋은 곡을 모티브로 받을 때 아래처럼 분해한다.

```text
Reference:
왜 좋은가:
감정 엔진:
후렴 작동 방식:
에너지 곡선:
보컬 거리:
편곡 질감:
절대 가져오지 않을 것:
Re:Version 변환 방향:
```

## Genre Lanes

장르를 다양하게 쓰되 곡마다 중심은 하나만 둔다.

| Lane | 잘 맞는 감정 | 주의점 |
|---|---|---|
| Korean dawn ballad | 미련, 고백, 후회 | 과한 신파 금지 |
| R&B / neo soul | 욕망, 거리감, 후회 | 멜리스마 과잉 금지 |
| city pop | 추억, 도시적 쓸쓸함 | 복고 장식만 남기지 말 것 |
| indie rock | 자조, 청춘, 무력감 | 기타 리프 복제 주의 |
| synthpop | 거리, 밤, 비현실감 | 감정이 차갑게만 남지 않게 |
| bossa / acoustic | 체념, 오래된 그리움 | 관광 이미지 금지 |
| trot / retro pop | 서러움, 체면, 해학 | 패러디처럼 만들지 말 것 |
| pansori-pop | 운명, 한, 선언 | 전통 요소를 장식품처럼 쓰지 말 것 |
| ambient electronic | 고독, 기억, 꿈 | 후렴 훅이 약해지기 쉬움 |
| cinematic pop | 클라이맥스, 큰 선택 | 예고편 음악처럼 과장 금지 |

## Song Quality Gate

0-3점으로 판정한다.

```text
후렴 기억도:
벌스 흡입력:
프리코러스 상승감:
브릿지 필요성:
장르 신선도:
보컬 적합성:
반복 청취 가능성:
원곡 독립성:
```

18점 미만이면 작사/Suno로 넘기지 말고 장르와 후렴 기능을 다시 잡는다.
