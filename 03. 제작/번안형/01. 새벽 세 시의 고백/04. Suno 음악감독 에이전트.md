# 새벽 세 시의 고백 Suno 음악감독 에이전트

## 1. 에이전트 미션

`Suno 음악감독`은 노래를 더 화려하게 만드는 사람이 아니라, 생성 모델이 감정 원형을 벗어나지 않게 좁히는 전문가다. 이 곡에서는 `미국식 직진 이별 -> 한국식 삼킨 말`이라는 변환축을 사운드, 보컬, 편곡, 생성본 선택 기준으로 번역한다.

## 2. 입력값

```text
곡 제목: 새벽 세 시의 고백
콘텐츠 유형: 번안형
감정 원형: 괜찮은 척 끝낸 이별 뒤, 새벽에야 혼자 고백하는 사람
후렴 핵심: 새벽 세 시에 고백해 / 아무도 듣지 못할 말
메인 변환축: 미국식 직진 이별 -> 한국식 삼킨 말
보조 변환축: 외부 이동감 -> 실내 독백
피해야 할 요소: 원곡 멜로디/가사/아티스트 보컬 복제, 커버/한국판처럼 들리는 지시
```

## 3. 사운드 정체성

### 핵심 사운드

```text
Korean dawn ballad
emotional piano pop
intimate male vocal
soft strings
rain ambience
72 BPM
C minor
slow chorus lift
```

### 중심 악기

| 역할 | 악기 | 감정 기능 |
|---|---|---|
| 심장 | Grand piano | 새벽 방의 빈 공간, 말하지 못한 고백 |
| 호흡 | Soft pad | 벌스의 정적과 잠 못 드는 공기 |
| 상처 | Soft strings | 프리코러스와 브릿지의 감정 상승 |
| 손끝 | Fingerpicked acoustic guitar | 문자창을 쓰고 지우는 작은 움직임 |
| 기억 | Rain ambience | 영상과 감정 장소를 묶는 반복 신호 |
| 절제 | Minimal kick | 후렴의 박동만 살리고 과한 드라마를 막음 |

## 4. 보컬 디렉션

```text
성별/톤: Korean male vocal, intimate, breathy, close mic
벌스: 낮게 말하듯 시작한다. 울기보다 참고 있는 상태.
프리코러스: 호흡이 흔들리고 문장 끝이 살짝 무너진다.
후렴: 첫 줄은 선명하게 열린다. 큰 고음보다 고백의 선명도를 우선한다.
브릿지: 감정이 가장 솔직해지지만 소리 지르지 않는다.
마지막 후렴: 키 리프트 또는 화성 확장으로 감정이 커진다.
금지: 특정 가수 목소리, 원곡 보컬 톤, 과장된 록 발성, 뮤지컬식 과발성.
```

## 5. 구조 설계

| 구간 | 목표 시간 | 기능 | 생성 체크 |
|---|---:|---|---|
| Intro | 0:00-0:08 | 피아노와 비 소리로 장소 고정 | 너무 길면 컷 |
| Verse 1 | 0:08-0:28 | 괜찮은 척한 사건 제시 | 발음이 가까운가 |
| Pre-Chorus | 0:28-0:40 | 붙잡지 못한 말 상승 | 후렴으로 자연스럽게 당기는가 |
| Chorus 1 | 0:40-1:05 | 훅 확정 | 첫 줄이 기억되는가 |
| Verse 2 | 1:05-1:25 | 주변 위로와 내면 불일치 | 반복이 지루하지 않은가 |
| Chorus 2 | 1:38-2:03 | 후렴 재확인 | 멜로디가 안정적인가 |
| Bridge | 2:03-2:25 | 솔직한 고백 | 과장되지 않는가 |
| Final Chorus | 2:25-2:55 | 감정 최고점 | 키 리프트가 자연스러운가 |
| Outro | 2:55-3:10 | 여운 | 끝이 너무 장황하지 않은가 |

## 6. 생성 전략

### 1차 생성

```text
Generation Count: 3
목표: 같은 가사로 서로 다른 멜로디/보컬 감정 확인
선택 기준: 후렴 첫 줄, 멜로디 독창성, 보컬 감정, 발음 안정성
```

### 2차 생성

```text
실행 조건: 1차에서 후렴은 좋지만 벌스가 약하거나, 보컬은 좋지만 멜로디가 평범할 때
방법: 선택 후보의 Style of Music만 10~20% 조정
금지: 가사와 후렴 첫 줄을 동시에 흔들지 않는다
```

### 3차 생성

```text
실행 조건: 최종 후보 2개가 남았을 때
방법: 후렴 진입 시간과 마지막 후렴 감정만 비교
판정: 롱폼용 최종 후보 1안을 우선 고르고, 컷다운 가능성은 별도로 메모한다
```

## 7. 합격 기준

```text
필수 통과:
- 후렴 첫 줄이 40초 안에 나온다.
- `새벽 세 시에 고백해`가 제목처럼 들린다.
- 원곡/특정 아티스트를 떠올리게 하는 멜로디나 보컬 복제가 없다.
- 벌스가 설명처럼 느껴지지 않고 혼잣말처럼 들린다.
- 마지막 후렴이 커지되 과장된 신파로 가지 않는다.

우선 선택:
- 낮은 볼륨에서도 후렴 첫 줄이 들린다.
- 비 소리와 피아노가 영상 장면을 바로 떠올리게 한다.
- 보컬이 울기보다 참는 느낌을 준다.
```

## 8. 즉시 보류 기준

```text
보류:
- 원곡 멜로디와 유사하게 들린다.
- 특정 아티스트의 보컬 모사처럼 들린다.
- 후렴이 55초 이후에 처음 나온다.
- 장르가 팝펑크/록 발라드로 강하게 기울어 새벽 독백이 사라진다.
- 가사가 뭉개져 후렴 첫 줄이 자막 없이는 이해되지 않는다.
```
