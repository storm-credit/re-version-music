# 새벽 세 시의 고백 Suno 실행 카드 - Take A

## 1. 실행 목적

이 문서는 Suno Custom Mode에 바로 붙여 넣기 위한 실행 카드다. 내부 영감 원천, 원곡명, 특정 아티스트명은 넣지 않는다. Take A는 `새벽 독백`, `피아노 발라드`, `절제된 남성 보컬`을 기준으로 3회 생성한다.

## 2. 실행 전 확인

- [ ] Suno Custom Mode를 사용한다.
- [ ] 아래 Title, Style of Music, Lyrics만 복사한다.
- [ ] 원곡명, 아티스트명, 커버/리믹스/한국판 표현을 추가하지 않는다.
- [ ] 생성본은 A-1, A-2, A-3으로 기록한다.
- [ ] 생성 후 `07. 생성본 선별 체크리스트.md`에 길이, 후렴 첫 진입, 총점을 기록한다.
